# AGENTS

## 開発コマンド
- `npm i`
- `npm run dev`
- `npm run build`
- `npm run lint`

## ディレクトリ方針
- `data` : データ取得や外部API連携などのデータ層
- `content` : コピーや文言、翻訳、静的アセットなどのコンテンツ層
- `ui` : 見た目やスタイルに関するUI層
- `core` : ドメインロジックや共通ユーティリティなどのコア層

## コーディング規約
- 可能な限り関数コンポーネントを利用すること
- 状態や副作用の管理はHookを介して行うこと
- 単方向データフローを守り、propsを通じてデータを受け渡すこと

## 変更時の確認手順
- ビルドが成功すること（`npm run build`）
- ルーティングが期待通りに動作すること
- モバイル表示でのレイアウト崩れがないこと
